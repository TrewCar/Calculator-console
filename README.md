# Calculator RU

Данынй калькулятор может работать с любым количеством заданных чисел, правда только с целыми числами, а не с дробными.

Первым вводится число элементов массива (только нечётное), тоесть все числа и знаки (только эти: / * + -), после этого вводятся все элементы поочерёдно через enter (число, знак, число)
